Acerca de S_CONTABLE

S_CONTABLE es un sistema integral de gestión contable diseñado específicamente para empresas colombianas. Permite administrar múltiples empresas desde un solo panel, gestionar catálogos, facturación, tesorería, contabilidad y reportes financieros con total cumplimiento de la normativa colombiana.

Sistema desarrollado con Django 5 • PostgreSQL • API REST
